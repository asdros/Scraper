# Scraper


The Web Api Scraper written in .Net Core 3.1.

Used technologies:
- Basic Auth
- Dapper
- Swashbuckle
- HtmlAgilityPack


DEMO: 
http://scrap-wsinf.somee.com/



For proper operation it is required to execute the script.sql in DBMS and change the "DefaultConnection" string in appsettings.json.
